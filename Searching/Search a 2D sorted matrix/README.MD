# 2D Array Search

Given a 2D sorted array. Find the target element. If it exists return its index.

### Example 1:

```
Input: array = [[1,3,5,7],[10,11,16,20],[23,30,34,60]] , target = 3
Output: [0,1]
```

### Example 2:

```
Input: array = [[1,3,5,7],[10,11,16,20],[23,30,34,60]] , target = 13
Output: [-1,-1]
```
